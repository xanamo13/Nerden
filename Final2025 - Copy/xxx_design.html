Intermediate Web SP2025

This is an e-commerce website for a fictional shop/company under the name "Nerden".

Audience

The intended audience is:

  • Users aged 13 and up, mainly 13-early 20s
  • Cartoon enthusiasts
  • Comic book enthusiasts
  • Collectors

Scope

This website covers what items are available at the shop, events the shop is going to hold, where to contact the shop, and how to apply to work there.

Goals

The company is looking to:

  • Increase sales by at least 6%
  • Increase store traffic by 5%
  • Increase visibility by 10%
  • Increase event attendance by 3%

Navigational Tools

  • Main Navigation Bar to navigate the main pages
  • Text Link Bar to navigate extra pages
  • Footer Links
  • Shop Aside/Nav Bar to browse products

Outline

I first thought of the audience. This helped me think of what information the user would most likely expect to see on the site. This also helped with the name and color palette.

Next, I made sure to look at what javascripts needed to be used, and how I can incorporate that into the fictional website. For the countdown, I thought of what kind of things the community would be looking forward to, like a new comic issue or an upcoming event. For the verification of a form, I thought of what the shop would need a form for. For example, a job, trade in, or event application.

Afterwards, I thought of the layout and what the user would most likely be interested in seeing. This included things like what the shop sells, or how to contact them. This is how I chose which links would be part of the main navigation bar, which ones would be in the text link bar, and what information should be displayed in a certain way.
